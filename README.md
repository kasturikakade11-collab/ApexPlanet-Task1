
# ApexPlanet Task 1 - Data Immersion & Wrangling

## Objective

The objective of this project is to perform data immersion, data quality assessment, data cleaning, and feature engineering to prepare the dataset for analysis.

## Dataset

Sales Dataset containing customer, product, and transaction information.

## Tasks Performed

### 1. Data Familiarization

* Explored dataset structure
* Examined columns and data types
* Reviewed dataset contents

### 2. Data Quality Assessment

* Checked for missing values
* Investigated duplicate Order_ID values
* Identified data inconsistencies

### 3. Data Cleaning

* Handled missing values in the City column
* Validated duplicate records
* Standardized dataset for analysis

### 4. Feature Engineering

* Created additional analytical features from existing data
* Prepared data for future reporting and analysis

### 5. Final Validation

* Verified cleaned dataset
* Confirmed dataset readiness for analysis
